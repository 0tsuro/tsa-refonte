# TSA - Site Web

Site construit avec PHP, CSS moderne, et JavaScript vanilla.

## Structure du projet

```
TSA/
├── assets/          # Images, fonts, icônes
├── config/          # Configuration (mail, reCAPTCHA)
├── css/             # Feuilles de style
├── en/              # Pages en anglais
├── fr/              # Pages en français
├── includes/        # Composants PHP réutilisables
├── js/              # Scripts JavaScript
├── php/             # Logique métier
└── uploads/         # Uploads utilisateurs
```

## Installation

1. Dupliquer `.env.example` en `.env`
2. Remplir les clés reCAPTCHA (optionnel)
3. Servir via PHP local ou déployer sur un serveur
